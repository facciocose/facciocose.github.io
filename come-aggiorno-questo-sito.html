Ecco qualche informazione su come aggiorno questo giardino digitale.

Scrivo le mie note utilizzando Obsidian. Ho scelto questo software perché mi permette di sincronizzare facilmente le note su linux, windows e il telefono. Per farlo utilizzo un estensione che si chiama Self-hosted LiveSync.

Per convertire i file markdown creati da Obsidian in file HTML utilizzo uno script python che ho creato un po' perché volevo che fosse facilmente personalizzabile e un po' perché mi andava di farlo. Si chiama obgage ed è disponibile su github.

I file HTML generati vengono inviati a un repository git. I push su questo repository avviano un deploy del sito su Cloudflare.

Questo processo richiede diverse operazioni manuali ma per il momento mi sta bene così.
